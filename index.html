HAL is not listening.

Say “OK HAL” to make him start listening.

Say “Goodbye HAL” to make him stop listening.

This project is an exploration of the Web Speech API, which allows voice recognition (speech to text), as well as speech synthesis (text to speech). More information can be found on the Google Developers site
